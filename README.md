# MetaTrader5-custom-Indicators
MetaTrader5 custom Indicators

- 20%_lines indicators
This indicator creates lines every 20% increase from base line figure with computed figures above the lines.
The purpose is to anchor the pricee line target in both upward and download directions in MT5 charts.
Every upward 20% increse (bull) is equal to 10% drcrease (bear) chanage

Four indicaors for HK50, ND100, SP500, and XAUXUSD are provided with best base line figures setup.
Base figures can be easily change in code or parameters menu for other symbols

- 5-200_Moving_Average Indicator
This indicator provides easy auto Moving Average lines for periods of 5, 10, 20, 50, 100, 200

- MACD indicator
Optimized version

- Stochastic Indicator
Optimized version

- Setup Steps
copy the files to /MQL5/Indicators folder
open the files with MetaQuotes Language Editor (F4)
Click Compile to generatel .ext files
drag the .ext file from Navigator to your charts of preferred symbols
